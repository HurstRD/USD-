Building JavaFX Webkit Source Code

Prerequisites

To build JavaFX Webkit you will need the following:

For All Systems

  • Install JDK version 6 or 7. Make sure java is in the PATH, and set JAVA_HOME to point to the location of the JDK installation.
  • Install qmake utility version 2.01a which is part of the Qt SDK version 4.6.0. Make sure qmake is in the PATH.

For Windows

  • Install Microsoft Visual Studio 10.0 compiler. Make sure to run vcvars32.bat to set up build environment.
  • Install Cygwin and the following additional packages:
    • bison
    • flex
    • g++
    • gperf
    • makedepend
    • perl
    • python
    • unzip
  • Set an environment variable:
    • QMAKESPEC=win32-msvc2008

For Mac

  • Install Xcode 3.2.1 Developer Tools
  • Set some environment variables:
    • QMAKESPEC=macx-g++
    • QMAKE_CFLAGS=-m64
    • QMAKE_LFLAGS=-m64

For Linux

Install the following packages:

  • g++ compiler version 4 or newer
  • bison
  • flex
  • gperf
  • libxml2-dev
  • libxslt1-dev
  • pkg-config
  • qt4-qmake

Building The Source

To set the desired build configuration, run from the top level directory

./Tools/Scripts/set-webkit-configuration [--debug|--release]

Start the build by running

./Tools/Scripts/build-webkit --java --imageio
