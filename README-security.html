WebView Security Features

WebView currently supports the following security-related features, as reported by the browserscope.org security test:

  • HTML 5 cross document messaging
  • JSON.parse
  • httpOnly cookie attribute
  • X-Frame-Options
  • Block location spoofing
  • Block JSON hijacking
  • Block XSS in CSS
  • Sandbox attribute
  • Block cross-origin CSS attacks
  • Cross Origin Resource Sharing
  • Block visited link sniffing
  • Content Security Policy
